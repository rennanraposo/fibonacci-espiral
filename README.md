# fibonacci-espiral
Será um projeto para a aula de "Desenvolvimento WEB" no qual pretendo criar uma aplicação web simples para demonstrar o funcionamento do cálculo de Fibonacci e a vizualização da espiral, conforme for clicando em um botão. Atendendo todos os requisitos dados pelo professor.

Separei o projeto em responsabilidades: o HTML cuida da estrutura, o CSS da aparência e o JavaScript da lógica. Isso deixa o código mais organizado, legível e fácil de manter.
